Somfy Protect Integration for Homey

Connect your Somfy Protect system to Homey.

Features
- Login with your own Somfy account during pairing
- Add IntelliTag, Motion Sensor, Siren, and Site Alarm devices
- View and control site alarm state (Disarm, Partial, Arm)
- Dashboard widgets for quick alarm control

Setup
1. Install the app on Homey
2. Add a Somfy device and sign in with your Somfy credentials
3. Devices are discovered and synced automatically

Notes
- This app uses cloud connectivity to Somfy Protect APIs
- Polling interval can be configured in app settings

Support
If you run into issues, check app logs and verify your Somfy account credentials.
